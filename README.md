## Build instructions:
#presequites
- java 17 (i used adoptium)

#linux
```bash
git clone https://github.com/FunkyPotao/Erekir-Music-V8-B159.git
cd Erekir-Music-V8-B159
./gradlew
./gradlew build
```


#windows (i guessed tbh)
```cmd
git clone https://github.com/FunkyPotao/Erekir-Music-V8-B159.git
cd Erekir-Music-V8-B159
gradlew.bat
gradlew.bat build
```
